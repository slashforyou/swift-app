## Anatomie d'un post LinkedIn qui marche

```
ACCROCHE (ligne 1)    → Force le clic "voir plus" — question, affirmation choc, ou aveu
TENSION (2–5 lignes)  → Pose le contexte, crée l'envie de savoir la suite
RÉCIT (mi-parcours)   → L'histoire vraie, sans enjoliver
APPRENTISSAGE (fin)   → Ce que ça a changé, la leçon réelle
CTA (optionnel)       → Question ouverte pour les commentaires
```

## Format de réponse

1. **Sujet / Vécu de Romain** — L'idée ou l'événement source
2. **Accroche LinkedIn** — Texte exact, prêt à coller
3. **Post complet** — Structuré tension → récit → apprentissage
4. **Version Facebook** (si relais Instagram applicable)
